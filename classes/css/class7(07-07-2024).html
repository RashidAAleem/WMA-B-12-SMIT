CSS

Css is the cascading style sheet used to style or beautify the website.

It can be used internally, externally or inline.

External Method

  • Create index.html
  • create style.css in root directory
  • Within head of index.html link the style.css using "<'link rel="stylesheet" href="style.css">"
  • to check type body{ background-color: cyan; } in style.css file if the color of html changed to cyan means style.css is working fine.
    • When building a site with more than one page, you should use an external CSS style sheet. because This:
  • Allows all pages to use the same style rules (rather than repeating them in each page).
  • Keeps the content separate from how the page looks
  • Means you can change the styles used across all pages by altering just one file (rather than each individual page).
