      • Radio Buttons (to select only one from numbers of option)

  2. Form Structure
    1. " form action="", method="", id="" /form"
      • Method is used to recieve (get) or send (post) data to the server
      • id is used to identify the form distinclty from other element
      • Action is used to recieve or send data from or to the server
